# Bloom Contraction Timer üå∏

**Bloom Contraction Timer** is a beautiful, easy-to-use contraction tracking application designed to support expecting parents during labor. With a focus on simplicity and calmness, it helps you track contraction duration, frequency, and intensity, providing real-time guidance on when to head to the hospital.

## ‚ú® Key Features

- **One-Tap Tracking**: Simply tap to start and stop contractions. No fuss when it matters most.
- **Smart Analysis**: Automatically calculates frequency and duration averages over the last hour.
- **Hospital Alerts**: specific guidance based on standard rules (like the 5-1-1 rule) to let you know when labor is progressing.
- **Milestone Logging**: Track important events like "Water Broke", "Mucus Plug", or "Left for Hospital".
- **Intensity Rating**: Record pain levels for each contraction to see trends over time.
- **Edit & Delete**: Fix mistakes easily if you tapped too early or late.
- **Dark Mode**:  Designed with a dark, low-light friendly interface perfect for night labor.
- **Cross-Platform**: Built for both Android and iOS.

## üì± Tech Stack

- **Framework**: [React Native](https://reactnative.dev/) with [Expo](https://expo.dev/) (Managed Workflow)
- **Language**: TypeScript
- **Backend / Auth**: [Supabase](https://supabase.com/)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Styling**: StyleSheet with custom design system
- **Routing**: Expo Router (File-based routing)
- **Icons**: Lucide React Native

## üöÄ Getting Started

### Prerequisites

- Node.js and npm
- Expo Go app on your phone (or a simulator)

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/your-username/contraction-timer.git
    cd contraction-timer
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the app**
    ```bash
    npx expo start
    ```

4.  **Run on device**
    - Scan the QR code with your phone (using Expo Go or Camera app).
    - Press `a` for Android Emulator or `i` for iOS Simulator.
